Политика обработки персональных данных

1. Оператор персональных данных

ИП [ФИО], ИНН [ИНН], ОГРНИП [ОГРНИП] (далее — «Оператор»), является оператором персональных данных в соответствии с Федеральным законом № 152-ФЗ «О персональных данных».

2. Состав обрабатываемых персональных данных

Оператор обрабатывает следующие персональные данные пользователей:

  • Фамилия, имя
  • Номер телефона
  • Адрес доставки (город, улица, дом, квартира)
  • Данные об оформленных заказах

3. Цели обработки

Персональные данные обрабатываются в следующих целях:

  • Исполнение договора (оформление и доставка заказов)
  • Связь с клиентом для подтверждения заказа
  • Улучшение качества сервиса

4. Правовое основание

Обработка персональных данных осуществляется на основании согласия субъекта персональных данных (ст. 6, 9 Федерального закона № 152-ФЗ) и в целях исполнения договора.

5. Передача третьим лицам

Персональные данные не передаются третьим лицам, за исключением случаев, предусмотренных законодательством РФ. Для целей доставки адрес может быть передан курьерской службе.

6. Хранение данных

Данные хранятся в течение срока, необходимого для достижения целей обработки, а также в течение срока, установленного законодательством РФ. Данные о заказах хранятся не более 3 лет.

7. Права субъекта персональных данных

Вы вправе:

  • Получить информацию об обработке ваших персональных данных
  • Потребовать исправления неточных данных
  • Отозвать согласие на обработку персональных данных
  • Потребовать удаления ваших данных

Для реализации прав обращайтесь через Telegram: @Reforma_request_bot

8. Безопасность

Оператор принимает необходимые организационные и технические меры для защиты персональных данных от несанкционированного доступа, изменения, раскрытия или уничтожения.
